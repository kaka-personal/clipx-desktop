# ClipX Desktop v0.1.0

首个公开版本，提供面向 Windows 的 ClipX 风格剪贴板管理体验。

## Highlights

- 托盘常驻与快速历史列表
- 文本、图片、文件列表剪贴板历史
- 常驻片段与快捷粘贴
- 设置面板与提示音
- 托盘图标切换
- Windows 安装版与便携版

## Downloads

- `ClipX-Desktop-Setup-0.1.0.exe`
- `ClipX-Desktop-0.1.0-Portable.exe`

## Notes

- 推荐普通用户优先下载安装版 `Setup.exe`
- 便携版适合不想安装的场景
- 用户数据默认保存在：
  `C:\Users\<用户名>\AppData\Roaming\clipx-desktop`
